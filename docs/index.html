Segurança de nível empresarial, API amigável para desenvolvedores

Detecção de Prompt Injection

Detecta sobrescritas de instruções, injeções de papel, ataques de delimitador, jailbreaks, vazamento de prompt e injeção codificada em 12 idiomas, alem de ataques multilíngues.

Detecção e Redação de PII

Encontra e redige automaticamente e-mails, SSNs, cartões de crédito (validados por Luhn), números de telefone, API keys, tokens do GitHub, chaves da AWS e números de passaporte.

Segurança de Uso de Ferramentas

Escaneia chamadas de ferramentas de agentes em busca de comandos perigosos, exfiltração de dados, acesso a credenciais e escalonamento de privilégios antes que sejam executados.

Filtragem de Conteúdo Nocivo

Bloqueia fabricação de armas, síntese de drogas, automutilação, criação de malware, discurso de ódio, phishing e instruções de fraude.

Integração com Claude Code

Hook PreToolUse plug-and-play, servidor MCP com 7 ferramentas, 5 slash commands e skill de segurança auto-invocada. Configuração com um único comando: sentinel init.

Red-Teaming Adversarial

O testador adversarial nativo gera variantes de evasão usando homóglifos, caracteres de largura zero, leetspeak e mais. Teste sua segurança antes que os atacantes o façam.

Segurança em Conversas de Múltiplos Turnos

Acompanha a segurança ao longo de conversas inteiras. Detecta escalonamento gradual de jailbreak, persistência de tópico, ataques sanduíche e novas tentativas após bloqueios.

Validação de Saída Estruturada

Escaneia JSON gerado por LLM em busca de XSS, SQL injection, injeção de template e path traversal ocultos em valores de campo. Validação de schema incluída.

Detecção de Ofuscação

Captura payloads de ataque codificados que driblam filtros de palavras-chave: base64, comandos codificados em hex, ofuscação ROT13, escapes unicode e variantes leetspeak de termos perigosos.

Scanner de Vulnerabilidades em Código

Escaneia código gerado por LLM em busca das vulnerabilidades do OWASP Top 10: SQL injection, injeção de comando, XSS, segredos codificados, desserialização insegura, criptografia fraca, SSRF e path traversal.

Firewall de API LLM

Proxy reverso transparente para qualquer API de LLM (Anthropic, OpenAI). Escaneia todas as requisições/respostas, bloqueia conteúdo perigoso em pleno streaming, redige PII automaticamente e impoe allowlists de modelos. Um comando: sentinel proxy.

Saída SARIF

Gera saída SARIF v2.1.0 para o GitHub Code Scanning, Azure DevOps e qualquer ferramenta de análise estática. Envie os resultados para a aba Security do GitHub com uma flag: upload-sarif: true.

Auditoria de Segurança

sentinel audit pontua a configuração de segurança do seu projeto (0-100). Verifica hooks do Claude Code, allowlist de permissões, arquivos .env, pre-commit hooks e configuração MCP, com sugestões de correção acionáveis.

Validador de Schema de Ferramentas MCP

Valida definições de ferramentas MCP em busca de vetores de injeção antes de confiar nelas. Detecta prompt injection, personificação de autoridade, exfiltração de dados e instruções de ocultação escondidas em descrições de ferramentas. sentinel mcp-validate.

Scanner de Dependências

Escaneia requirements.txt, package.json e pyproject.toml em busca de ataques à cadeia de suprimentos. Detecta typosquatting, pacotes maliciosos conhecidos, URLs suspeitas e scripts de instalação perigosos. sentinel dep-scan.

Scanner de CLAUDE.md

Escaneia arquivos de instrução do projeto (CLAUDE.md, .cursorrules) em busca de 11 categorias de vetores de injeção: comentários HTML ocultos, personificação de autoridade, sobrescrita de base URL, caracteres de largura zero e mais. sentinel claudemd-scan.

Scanner de Segredos

Detecta API keys, tokens e credenciais codificados em mais de 40 provedores: AWS, GitHub, Google, Stripe, Slack, OpenAI, Anthropic e mais. Análise de entropia e filtragem inteligente reduzem falsos positivos. sentinel secrets-scan.

Canary Tokens

Planta marcadores invisíveis em system prompts. Se aparecerem na saída do modelo, seu prompt vazou. Dois estilos: canaries em comentário HTML e codificação Unicode de largura zero (verdadeiramente invisível).

Hardening de Prompt

Torna system prompts resistentes a injeção com defesa em profundidade: marcação de seções em XML, defesa sanduíche, bloqueio de papel, marcadores de prioridade de instrução e isolamento de dados não confiáveis.

Mapeador de Compliance

Mapeia resultados de escaneamento para controles do EU AI Act, NIST AI RMF e ISO/IEC 42001. Classificação de risco automatizada, status de conformidade por controle e recomendações de remediação para relatórios regulatórios.

Monitor de Segurança de Agentes

Acompanha padrões de uso de ferramentas em sessões de agentes. Detecta comandos destrutivos, exfiltração de dados, acesso a credenciais, loops descontrolados, picos de escrita e cadeias de ataque de leitura-seguida-de-exfiltração em tempo real.

Wrappers de Guard para SDKs

Integração em uma linha para os SDKs da Anthropic e da OpenAI. guard_anthropic(client) envolve seu client com escaneamento automático de entrada/saída. Lança exceção em ataques de injeção, vazamentos de PII ou conteúdo inseguro.

Feed de Threat Intelligence

Base de dados alinhada ao MITRE ATLAS com mais de 27 técnicas conhecidas de ataque a LLMs. Consulte por categoria, severidade ou tags. Compare texto com os padrões. Cobre injeção, jailbreak, exfiltração, evasão e mais.

Detector de Cadeia de Ataque

Detecta sequências de ataque em multiplas etapas entre chamadas de ferramentas. Nenhuma ação isolada e perigosa, mas a sequência revela intenção maliciosa: reconhecimento → acesso a credenciais → exfiltração, escalonamento → destruição, e mais.

Session Guard

Segurança em uma linha para IA agêntica. Combina audit logging, detecção de cadeia de ataque e threat intelligence em uma única chamada guard.check(). Limiar de bloqueio configurável, regras customizadas e exportação JSON pronta para SIEM.

Trilha de Auditoria de Sessão

Logging a prova de adulteração com encadeamento de hash SHA-256. Toda chamada de ferramenta, ação bloqueada e anomalia e registrada. Exporte para JSON para Splunk, Datadog, Elastic. Pronto para compliance com SOC 2 e ISO 27001.

Recursos Enterprise

Motor de políticas (YAML), webhooks (Slack, PagerDuty), OpenTelemetry, autenticação por API key, rate limiting, proteção de streaming, relatórios de risco alinhados a RSP.
