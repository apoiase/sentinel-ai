## O que é

Biblioteca Python (`sentinel-guardrails` no PyPI) + SDK TypeScript (`sdk-js/`, no npm) de
guardrails de segurança para aplicações LLM — scanners determinísticos (regex, sem GPU/API
externa) de prompt injection, PII, conteúdo nocivo, alucinação, toxicidade, código vulnerável,
segredos etc. Origem: fork traduzido pra pt-BR de um projeto open source de terceiro
(`MaxwellCalkin/sentinel-ai`), mesclado via PR #1 já aberto pela apoiase. Não é um serviço da
Apoia.se — é uma ferramenta/produto adotado (`ACQUISITION_PITCH.md`, `FOLLOWUP_EMAIL_DRAFT.md`
sugerem avaliação de aquisição/parceria em andamento, fora do escopo deste scan).

## Deploy real: distribuição de pacote, não runtime AWS

- `pyproject.toml` → build `hatchling`, publica `sentinel-guardrails` no PyPI via
  `.github/workflows/publish.yml` (trigger `release`/`workflow_dispatch`; usa
  **Trusted Publishing OIDC** — `permissions: id-token: write` + `pypa/gh-action-pypi-publish`,
  **sem token estático** — boa prática já presente).
- `sdk-js/` → publica no npm via `.github/workflows/publish-npm.yml` (usa
  `secrets.NPM_TOKEN`, token estático clássico do npm — não há OIDC/Trusted Publishing pro
  registry do npm ainda amplamente disponível para todos os pacotes, então isso é esperado).
- `Dockerfile` na raiz — builda uma imagem pra rodar `sentinel serve`/`proxy` localmente;
  **sem orquestração** (sem k8s manifests, sem ECS task def, sem docker-compose) — é
  self-host opcional do consumidor da lib, não deploy operado pela Apoia.se.
- `docs/index.html` + `site/index.html` — demo estática (link do README:
  `https://maxwellcalkin.github.io/sentinel-ai/`) — GitHub Pages do repo **upstream**
  (`MaxwellCalkin`), não deste fork.
- `.claude-plugin/` + `commands/` + `hooks/` — plugin do Claude Code distribuído via
  marketplace de plugins (`.claude-plugin/marketplace.json`), não infra AWS.

**Conclusão AWS: sem compute deployado.** Repo é biblioteca/tooling distribuída via PyPI + npm +
plugin marketplace — não há ECS/EKS/Lambda/EC2 a validar.

`docs/prc/security/PRC.md` (SEC4). Não rotacionado (não há nada pra rotacionar — são exemplos).
